# desktop-config-payload

TV desktop files live inside this module under `files/<intent>/...`. The folder names are the thought/intent: `hyprland`, `kitty`, `waybar`, `wofi`, `dunst`, `gtk`, `kate`, `kde-applications`, `systemd-user`, `browser`, `launcher-bin`, `shell-rc`, `prompt`, and `portal`.

Do not create sibling `profiles/tv/config` payload folders. Profile files belong in the module that owns the intent.
